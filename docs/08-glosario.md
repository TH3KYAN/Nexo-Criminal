# 📖 Glosario

| Término | Definición |
|---|---|
| **Red Thread / Hilo Rojo** | Vínculo descubierto automáticamente por el sistema entre dos o más entidades. Es la metáfora central del proyecto. |
| **Nodo** | Cualquier entidad representada en el grafo (persona, vehículo, ubicación, suceso). |
| **Arista / Vínculo** | Relación entre dos nodos. Puede ser explícita (propietario de) o descubierta (patrón de escolta). |
| **Nodo Logístico** | Ubicación identificada como probable centro de desmantelamiento de vehículos robados. |
| **Vehículo de Apoyo / Escolta** | Vehículo que aparece repetidamente cerca de vehículos robados antes de su desaparición. |
| **Intermediario Crítico** | Persona que conecta socialmente a una víctima con un sospechoso en un grado de separación bajo. |
| **Punto de Contacto (POI)** | Ubicación geográfica donde confluyen múltiples víctimas antes de perder señal. |
| **Modus Operandi** | Método delictivo usado en un suceso. |
| **Patrón de Banda** | Conjunto de sucesos agrupados por similitud de modus operandi. |
| **MVP** | Producto Mínimo Viable — versión más pequeña del sistema que entrega valor real. |
| **Sprint** | Iteración corta (1 semana en este proyecto). |
| **Historia de Usuario (HU)** | Descripción breve de una funcionalidad desde la perspectiva del usuario final. |
| **Definition of Done (DoD)** | Criterios que una historia debe cumplir para considerarse terminada. |
| **Backlog** | Lista priorizada de todo lo que se podría construir. |
| **PostGIS** | Extensión de PostgreSQL para datos geográficos. |
| **BFF (Backend for Frontend)** | Capa intermedia que adapta datos del backend para el frontend. |
| **JWT** | JSON Web Token — estándar de autenticación basada en tokens. |
| **Scrum** | Marco de trabajo ágil basado en sprints y ceremonias definidas. |
| **Planning Poker** | Técnica de estimación grupal usando cartas con valores Fibonacci. |
| **Velocidad (Velocity)** | Puntos que el equipo completa por sprint. |
| **Burndown** | Gráfica que muestra el trabajo restante en el sprint. |
| **Retrospectiva** | Reunión al final del sprint para mejorar el proceso. |
